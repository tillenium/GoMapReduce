# Map Reduce in Go

### Installation
```shell
make build
```

### Running instructions
```shell
./build/bin/gomr <.so file with Map/Reduce operation> <files>

#example:

./build/bin/gomr examples/word_count.so data/file1.txt data/file2.txt
```
